main module¶

main.main(option, datapath1, datapath2='')¶
Description:

  • Main entry point for the “Archer DX” interview library. You can use any of the below options to run a specific set of data or leave it all blank and it will run everything on its own

Default Options (solves Archer DX interview questions):
  • Part 1 (python main.py Part1 providedData/fastq)

    • option = “Part1”

    • datapath = “providedData/fastq”

  • Part 2 (python main.py Part2 providedData/fasta/sample.fasta)

    • option = “Part2”

    • datapath = “providedData/fasta/sample.fasta”

  • Part 3 (python main.py Part3 providedData/annotate/coordinates_to_annotate.txt, providedData/gtf/hg19_annotations.gtf)

    • option = Part3

    • datapath1 = providedData/annotate/coordinates_to_annotate.txt

    • datapath2 = providedData/gtf/hg19_annotations.gtf

-Alternatively

  • Just put in (python main.py) and the it will load the default values, which are the keys for the interview project
