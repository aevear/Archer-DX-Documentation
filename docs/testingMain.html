testingMain module¶

testingMain.main(option)¶
Description

  • Main testing suite. Runs a smaller version of the project and compares it to the Key. If the values are the same, lets the user know there are no inconsistencies.

Parameters

option (-) – The specific module that is being tested. Full is an acceptable response that runs all three.
