archerDX.utility.dataExport module¶

archerDX.utility.dataExport.outputResults(name, data)¶
Description:

  • exports the data with provided name to the results folder in tab delimited CSV format

Parameters

  • name (-) – name of the file that will be created

  • data (-) – results dataframe with the output for one of the other modules

archerDX.utility.paths module¶

Configuration Values

Module contents¶

Imports modules for data exporting and importing as well as the paths for the project

  • Data Export : Single function used to export the results of the project to the results folder found in data

  • Data Import : Imports a variety of file types and with different specifications for some of them

  • Paths : Pathing used by the rest of the code. uses the pathlib module
